Hello Shopify team! Thank you for taking the time to review my submission for the backend developer intern role. I'm very excited for you to evaluate my work!

On this page I've provided instructions on how this web app works. This app functions as a photo repository as well as a simple marketplace for photos (users can upload and sell pictures). I chose to use the Django framework in Python because I figured that it has a lot of plugins available which would reduce the lines of code in the backend, as well as the fact that I wasn't too comfortable with this framework when I started and I wanted to practice! (Also please note that I spent most of my efforts on the backend because of the job description, I don't have too much experience in UI/UX but I plan on getting there!)

To get started, you can create a username and password (email is required). Then, you can navigate to the home page, where you will see displayed in a 3x10 grid all of the photos on the site, in reverse chronological order. At any point, you can click the photo name to view details of the photo, or click the username of the user who uploaded the photo to view all photos that that user has uploaded.

Each upload is an instance of the django model "Photo", which can be found under "src/store/models.py". You'll notice that a Photo will have many attributes:
- Title
- Author (which is the user who uploaded the photo)
- Thumbnail (which is a scaled-down version of the photo used in the UI so that the server doesn't have to load larger photos all the time)
- Price
- Description
- Available (which indicates whether or not the Photo is currently listed for purchase)
- Featured (which indicates whether the site has decided to feature the Photo on the featured page. In a complete version, the site would construct an algorithm which allows certain photos to be featured at a time depending on what's trending at the moment.)
- Date Posted

Overall there are three objects in this web app demo: User, Photo and Cart. If at any time you want to check out the database to get a better understanding of how these objects interact with each other, I've created a user with admin rights for you to check it out!

Just click HERE and log in using:
User = shopify

(To return to the site, click "view site" at the top of the admin nav bar.)

A user may also look at their profile and update their information (including a profile picture; a default picture is always provided). A user may also make updates to any Photo that they have listed, including de-listing it, changing the price, the name, the description, or removing the Photo altogether.

Finally, when a user is done shopping, they make check out by using the navigation to the cart page (using the nav bar at the top of the page) and then clicking "check out". (Note that in this demo, there is no function implemented for actually purchasing the Photo(s) in the user's cart. Checking out will just return the user's cart total and then clear the cart.)
